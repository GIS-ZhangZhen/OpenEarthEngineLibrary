asImageToAsset()

oeel.Export.ImageCollection.asImageToAsset(imageCollection, description, assetId, pyramidingPolicy, dimensions, region, scale, crs, crsTransform, maxPixels)
asImageToAsset creates a batch task to export an ImageCollection as an Image raster to an Earth Engine asset. Tasks can be started from the Tasks tab. each image need to have the same bands

Arguments:

  • imageCollection The imageColelction to export as single Image.

Return:

  • Return (Image).

How to use?

To use this library simple copy past the following line you GEE:

var oeel=require('users/gravey_mathieu/testlibGithub_v2:openEEL')
